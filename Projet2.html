Site internet Lego

Présentation :


Ce projet a été réalisé en groupe de 2 sur une période de 2 mois. Il fallait répondre aux besoins d'un client en l'occurrence un de nos professeurs qui souhaitait un site vitrine de Lego

Image :


Description des différentes pages :


  • La page d'accueil :

    C'est la page principale de notre site, elle permet d'accéder à toutes les autres pages, cette page a été réalisée durant la période de Noël, elle contient donc des animations et des produit exclusif à Noel.

  • La page Notre histoire :

    Cette page rassemble l'histoire de Lego dans un livre, on peut tourner les pages grâce aux flèches sur le clavier.

  • La page Nos produit :

    C'est une page où on peut voir des Lego de trois univers différents (Star war, Technique, et Marvel). Chaque Lego a une description et une vidéo le présentant sous tous les angles.

  • La page Les records :

    Cette page permet de voir les différents record battu en Lego.
